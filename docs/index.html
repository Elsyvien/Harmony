Overview

What is Harmony?

Authenticated real-time chat with public channels, friend-gated direct channels, voice signaling, and admin runtime controls.

Entry Files

Backend: backend/src/server.ts & backend/src/app.ts
Frontend: web/src/main.tsx & web/src/App.tsx

Quick Start

Action Command
Install dependencies npm install
Apply schema npm --workspace backend exec prisma db push
Seed data npm --workspace backend run prisma:seed
Start full stack npm run dev
Run tests npm test
Run lint npm run lint
